# Project Architecture

## Overview

Contoso Retail Performance Management is an end-to-end Business Intelligence solution developed in Power BI using Microsoft's Contoso Retail DW dataset.

The dashboard simulates a Retail Performance Management system designed to support management decision-making through the analysis of revenue, profitability, margin leakage, geographic performance and store efficiency.

---

## Objectives

The project aims to:

- Monitor revenue and profitability
- Analyze product and category performance
- Identify margin leakage caused by discounts and returns
- Understand profitability drivers
- Evaluate geographic performance
- Measure store efficiency

---

## Dashboard Structure

The report is organized into six analytical areas:

1. Executive Overview
2. Sales & Margin Analysis
3. Discount & Return Analysis
4. Margin Drivers Analysis
5. Geographic Performance
6. Store Performance

---

## Technology Stack

- Power BI Desktop
- DAX
- SQL Server Express
- Microsoft Contoso Retail DW
