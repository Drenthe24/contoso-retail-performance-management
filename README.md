# Contoso Retail Performance Management

## Overview

Contoso Retail Performance Management is an end-to-end Business Intelligence solution developed in Power BI using Microsoft's Contoso Retail DW dataset.

The project simulates a Retail Performance Management system designed to support management decision-making through the analysis of revenue, profitability, operational efficiency and business drivers.

The dashboard combines financial KPIs, profitability analysis, margin leakage monitoring, geographic insights and store efficiency evaluation into a single analytical platform.

---

## Objectives

The main objectives of the project are:

* Monitor revenue and profitability performance
* Analyze product and category contribution
* Identify margin leakage caused by discounts and returns
* Understand profitability drivers
* Evaluate geographic performance
* Measure store efficiency and operational effectiveness

---

## Dashboard Structure

The report is organized into six analytical areas:

### 1. Executive Overview

Provides a high-level view of business performance through key financial and operational KPIs.

Main metrics:

* Revenue
* Gross Margin
* Gross Margin %
* Units Sold
* Discount Rate
* Return Rate

![Executive Overview](screenshots/01_overview.png)

---

### 2. Sales & Margin Analysis

Analyzes revenue generation and profitability by product category, subcategory and product.

Main features:

* Revenue analysis
* Cost analysis
* Margin analysis
* Product profitability
* Revenue contribution

![Sales & Margin Analysis](screenshots/02_sales_margin_analysis.png)

---

### 3. Discount & Return Analysis

Focuses on identifying profitability erosion caused by discounts and product returns.

Main features:

* Discount Rate analysis
* Return Rate analysis
* Margin Leakage monitoring
* Product-level leakage identification

![Discount & Return Analysis](screenshots/03_discount_return_analysis.png)

---

### 4. Margin Drivers Analysis

Explains the factors influencing gross margin performance.

Main features:

* Waterfall analysis
* Margin decomposition
* Cost incidence analysis
* Revenue contribution analysis

![Margin Drivers Analysis](screenshots/04_margin_drivers_analysis.png)

---

### 5. Geographic Performance

Evaluates business performance across countries and sales territories.

Main features:

* Geographic revenue distribution
* Country contribution analysis
* Regional profitability analysis
* Territory performance comparison

![Geographic Performance](screenshots/05_geographic_performance.png)

---

### 6. Store Performance

Analyzes store efficiency and operational performance.

Main features:

* Revenue per Store
* Revenue per Employee
* Revenue per Square Meter
* Store format comparison
* Operational efficiency analysis

![Store Performance](screenshots/06_store_performance.png)

## Data Model

The solution is based on a dimensional model built around the FactSales table.

Main tables:

### Fact Table

* FactSales

### Dimensions

* DimDate
* DimProduct
* DimProductSubcategory
* DimProductCategory
* DimStore
* DimGeography
* DimSalesTerritory

![Data Model](screenshots/data_model.png)

## Key KPIs

| KPI                  | Description                                  |
| -------------------- | -------------------------------------------- |
| Revenue              | Net revenue generated by retail transactions |
| Gross Margin         | Revenue minus Cost of Goods Sold             |
| Gross Margin %       | Profitability ratio                          |
| Margin Leakage       | Discounts and returns impact                 |
| Revenue Share %      | Contribution to total revenue                |
| Revenue per Employee | Workforce productivity                       |
| Revenue per Sqm      | Space utilization efficiency                 |
