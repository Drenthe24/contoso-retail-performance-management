# Contoso Retail Performance Management

This project is an end-to-end Business Intelligence solution developed in Power BI using Microsoft's Contoso Retail DW dataset.

The dashboard simulates a Retail Performance Management system designed to support management decision-making through the analysis of revenue, profitability, operational efficiency and business drivers.

## Key Features

- Executive Performance Overview
- Sales & Margin Analysis
- Discount & Return Analysis
- Margin Drivers Analysis
- Geographic Performance
- Store Performance & Efficiency

## Technologies

- Power BI
- DAX
- SQL Server
- Dimensional Data Modeling
- KPI Design & Business Analytics

## Business Focus

The project focuses on transforming raw retail data into actionable business insights, highlighting:

- Revenue and profitability trends
- Margin leakage caused by discounts and returns
- Product and category performance
- Geographic revenue distribution
- Store efficiency and operational performance
- Margin contribution drivers

This project demonstrates the application of Business Intelligence techniques, financial analysis concepts and performance management methodologies in a retail environment.
